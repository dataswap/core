## Usage

### Build
##### 1 dependencies
```shell
yarn install

```

2 compile
```shell
forge build

```
or
```shell
yarn hardhat compile
```

### Test

```shell
forge test
```
or
```shell
yarn hardhat test
```

### Gas Snapshots

```shell
forge snapshot
```

### Cast

```shell
cast <subcommand>
```
